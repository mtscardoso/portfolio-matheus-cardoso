# 🐋 OrçaPro — Budget Workspace

## 📝 Descrição do Projeto
O **OrçaPro** é uma plataforma web voltada para gerenciamento de clientes e criação de orçamentos profissionais. O sistema foi desenvolvido com foco em produtividade, organização comercial e acompanhamento do pipeline de propostas em tempo real.

A aplicação permite cadastrar clientes, criar novos orçamentos, acompanhar status de aprovação, visualizar métricas comerciais e gerenciar todo o fluxo de propostas através de um dashboard moderno e intuitivo.

O projeto foi desenvolvido utilizando a plataforma **Bubble.io**, com foco em arquitetura visual, responsividade e experiência do usuário, simulando um ambiente SaaS profissional para gestão financeira e comercial.
